About Me Section

I'm a senior at New College of Florida graduating in May 2025 with a Bachelor’s in Computer Science. I also hold an Associate of Arts from Hillsbrough Community College. With this academic foundation, I bring a unique blend of technical expertise and general knowledge that is supported by hands-on experience in software development, embedded systems, game development, and machine learning/data science.

My software development portfolio includes an animal shelter management platform, a chat web application that is similar to discord both built with JavaScript, Node.js, and Vite, and a “Stress-Free Calendar” web app created using Python and Streamlit. In the embedded systems space, I developed a red-light traffic control system in C++ using an Arduino Uno microcontroller. My game development experience includes the Unity games, a casino simulation written in Java using object-oriented design principles, and VR simulations through Oculus.

My senior thesis focused on building a centralized, interactive fishing dashboard aimed at improving how fishing data is collected, visualized, and analyzed. Developed using Streamlit, the dashboard integrates an interactive Folium map, Plotly visualizations, and real-time weather data via API calls. Users can log fishing trip details, track species trends, and gain insights through an intuitive interface. Designed for a broad audience—including marine biologists, conservationists, and both recreational and professional fishermen—the dashboard supports sustainable fishing practices and data-driven decision-making. The project reflects my lifelong passion for fishing and demonstrates my ability to combine technical development with real-world problem-solving.

Outside the classroom, I’ve played college baseball throughout my time at New College. This experience has shaped my leadership, time management, and ability to perform under pressure. I’m passionate about solving complex problems, learning new technologies, and building tools that are both functional and user-centered.
